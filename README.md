Luigi's World (v2)
===========

![Screenshot](screenshot.jpg)

An homebrew game written in Lua for PSP 


##What's this ?
A minimalistic platformer game.

##Notes
Compatible with PSP Slim and Fat (Uses *LPHM 5*).

##Installation Path
ms0:/PSP/GAME/
